# Reproducibility — HEP decay readout note

All numerics quoted in *Heavy-Flavour Branching Ratios Derived from Discrete Three-Ledger Rules on the HQIV Null-Lattice Carrier* are generated by the bundled Python reproducers. PDG and facility values are confined to a **quarantined comparison layer**; they never feed partial-width formulas or open-flavour seed priors. The pipeline regenerates the benchmark and readout export **before** comparison-layer JSON is consulted for pass/fail scoring.

## One-command reproduction (full paper numerics)

From the **repository root** (recommended when building the PDF):

```bash
export PYTHONPATH=papers/hep_decay_readout/scripts
python3 papers/hep_decay_readout/scripts/hqiv_hep_readout_pipeline.py paper --strict
```

From the **scripts bundle** (unzip `scripts.zip` or use `papers/hep_decay_readout/scripts/`):

```bash
cd scripts
export PYTHONPATH=.
python3 hqiv_hep_readout_pipeline.py paper --strict
```

This regenerates:

| Artifact | Role |
|----------|------|
| `data/hep_decay_benchmark.json` | **81** structural witness passes, **567** open-channel readout rows, **0** failures |
| `data/hep_decay_observations.json` | Comparison-layer definitions (**17** curated branching diagnostics) |
| `data/spine_discharge_law.json` | Unified spine product registry |
| `data/spine_gap_closure_terms.json` | Gap-closure routing diagnostics |
| `papers/hep_decay_readout/generated/*.tex` | Machine-generated tables in the PDF |
| `data/hep_readout_pipeline_manifest.json` | Step log (`ok: true` when strict run succeeds) |

Canonical write target is **repo-root** `data/`. The copy under `papers/hep_decay_readout/data/` mirrors the same schema for Zenodo/scripts-only bundles.

## Fast verification (< 30 seconds)

Confirms the frozen benchmark contract without rebuilding the full panel:

```bash
cd papers/hep_decay_readout/scripts   # or unzipped scripts/
export PYTHONPATH=.
python3 -m unittest test_hqiv_hep_smoke -q
```

Checks: **81** structural passes / **0** failures, **17** curated diagnostic definitions, flagship $J/\psi\to e^+e^-$ branching $\approx 5.97\%$.
